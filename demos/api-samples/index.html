Microsoft Quick Authentication samples

The following HTML pages are samples for the Microsoft Quick Authentication library. Each sample demonstrates a single feature or API method.

To see a more realistic example showing how to use these methods together, see our demo app.

For more information about the Microsoft Quick Authentication library, check out Sign in Microsoft account users to a single-page app (SPA) with Microsoft Quick Authentication.

  • Sign-in button with the <div class="ms-auth-button"> HTML code
  • Sign-in button with ms.auth.renderSignInButton
  • Customize the sign-in button
  • Programmatic sign-in with ms.auth.startSignIn
  • Sign-out with ms.auth.signOut
  • Get current account information with ms.auth.startGetCurrentAccount
  • Programmatically trigger the Microsoft Edge enhanced sign-in prompt with ms.auth.prompt
  • Use MSAL.js functionality by retrieving the MSAL account with ms.auth.getMSALAccount
  • Fetch MSAL tokens silently with ms.auth.acquireTokenSilent
  • Fetch MSAL tokens by opening a popup in the browser with ms.auth.acquireTokenPopup
